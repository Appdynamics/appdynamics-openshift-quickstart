Welcome to OpenShift - AppDynamics

Ensure you complete the following steps in order to integrate your application with AppDynamics.

From your local git repository, update your controller-info.xml file located in AppServerAgent/conf to include your controller-host, account-name and account-access-key. This information was located in your email after signing up for an AppDynamics account. You may also specify application-name, tier-name and node-name in this file. After updating these fields, run the following: 

    git commit -a -m 'Some commit message'
    git push

Go to your controller-host and your application should be listed.

See the README.md file in the application Git repository for more information on the options for deploying applications.
